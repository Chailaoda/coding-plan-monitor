# 安装与使用说明

本程序用于在 Linux/UOS 桌面显示火山方舟 Coding Plan 与小米 MiMo Token Plan 的用量悬浮窗。

## 一、安装依赖

```bash
git clone <your-repo-url>
cd coding-plan-monitor
python3.11 -m pip install -r requirements.txt
```

## 二、快速验证

如果你使用 360 浏览器 UOS 版，并且已经登录过火山方舟控制台和 MiMo 平台，可直接运行：

```bash
python3.11 test_api.py
```

再启动悬浮窗：

```bash
python3.11 main.py
```

## 三、自动 Cookie 模式

默认配置为：

```json
{
  "auth": {
    "mode": "auto"
  }
}
```

程序会读取：

```text
~/.config/com.360.browser/Default/Cookies
```

并在本机解密对应平台 Cookie。

## 四、手动 Cookie 模式

如自动模式不可用：

```bash
cp config.example.json config.json
chmod 600 config.json
```

将 `auth.mode` 改为 `manual`，并按 `cookie导出指南.md` 填入 Cookie。

```json
{
  "auth": {
    "mode": "manual",
    "cookie": "火山方舟完整 Cookie",
    "csrf_token": "<CSRF_TOKEN>",
    "mimo_cookie": "MiMo 完整 Cookie"
  }
}
```

## 五、悬浮窗操作

| 操作 | 效果 |
|------|------|
| 拖动标题栏 | 移动窗口 |
| 点击 `▼` / `▶` | 展开 / 折叠 |
| 点击 `✕` | 隐藏窗口（仅隐藏，托盘仍在；可点托盘图标重新唤出） |

折叠态示例：

```text
▶ 火山 5小时:49.5% 周:10.1% | MiMo:62.1%
```

## 五、A 系统托盘图标

启动后任务栏右下角会出现两个圆形进度环图标：

| 图标 | 含义 |
|------|------|
| 第一个 | 火山方舟（5h/周/月 中最高百分比） |
| 第二个 | MiMo 总用量百分比 |

颜色：<60% 绿、60%~90% 黄、≥90% 红、无数据灰（中央问号）。

| 操作 | 效果 |
|------|------|
| 鼠标悬停 | 显示英文 tooltip（含百分比） |
| 左键单击 | 切换悬浮窗显示/隐藏 |
| 右键单击 | XEmbed 后端不支持菜单，无效果 |

## 六、提醒规则

| 平台 | 阈值 | 行为 |
|------|------|------|
| 火山 5 小时 | 70 / 90 / 98 | 通知，critical 时铃声和闪烁 |
| 火山周 | 80 / 95 | 通知，critical 时铃声和闪烁 |
| 火山月 | 80 / 95 | 通知，critical 时铃声和闪烁 |
| MiMo | 60% 起，每跨越 5% | 通知；90% 起 critical |

## 七、systemd 用户服务

```bash
mkdir -p ~/.config/systemd/user
cp coding-plan-monitor.service.example ~/.config/systemd/user/coding-plan-monitor.service
```

编辑 service 文件，把：

```ini
ExecStart=/usr/bin/env python3.11 /path/to/coding-plan-monitor/main.py
```

改为你的实际项目路径。

启用：

```bash
systemctl --user daemon-reload
systemctl --user enable --now coding-plan-monitor
```

查看状态：

```bash
systemctl --user status coding-plan-monitor
journalctl --user -u coding-plan-monitor -f
```

停止/禁用：

```bash
systemctl --user disable --now coding-plan-monitor
```

## 八、隐私与安全

不要提交：

- `config.json`
- `usage.db*`
- Cookie、token、JWT、API Key、账号 ID

本项目的 `.gitignore` 已排除常见敏感和运行时文件。

## 九、故障排查

### 火山提示未找到 csrfToken

打开浏览器访问火山方舟控制台并刷新页面，等待 60 秒自动恢复。

### MiMo 无数据

确认浏览器已登录 MiMo 平台，并打开过订阅管理页。

### 悬浮窗不显示

```bash
python3.11 main.py
```

查看终端输出，检查 DISPLAY / X11 环境。
